# Vibisha Abhirami V Portfolio

A formal dark-themed portfolio website for Vibisha Abhirami V, built with React, TypeScript, and Vite.

## Live Site

https://vibisha23.github.io/portfolio/

## Sections

- Hero introduction
- About
- Areas of interest
- Technical skills
- Featured projects
- Education and certifications
- Contact

## Tech Stack

- React
- TypeScript
- Vite
- CSS
- GitHub Pages

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
